Blog on what is information

One thing i really enjoyed learning in Unit 1 was markdown. Being a new topic it was tricky in the begining but then i slowly got the hang of it during the lab classes. when i first started learning about markdown it was a little challenging but it was interesting. it gave a new perspective to how we write new documents. Using specific symbols instead of the plain old menu and buttons. Even though it was a new topic i could relate it to some things i already knew. it sort of reminded me of programming where symbols give outputs. in the same way in markdownwe use characters like hash or asteriks to control the text.

one challenge i faced you could say is remembering all the symbols and how they are used, that took some time to get used to. at first it was easy cause there were so little but as i learnt more it got a little challenging. What really helped me understand was if i took each topic step by step like focusing on headings first. learning it like this made it a lot easier and less overwhelming.
